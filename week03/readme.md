# 📦 Proyecto Semana 3 - Calculadora de Dominio (Gestión de Cementerios)

Este proyecto corresponde al único entregable obligatorio de la **Semana 3**.  
El objetivo es construir una **calculadora multi-operación** aplicada al dominio **Gestión de Cementerios**, usando operadores aritméticos, de asignación compuesta, de comparación estricta y lógicos.

---

## 🎯 Objetivos del Proyecto
- Calcular totales, descuentos y promedios con operadores aritméticos.
- Actualizar valores con operadores de asignación compuesta.
- Validar condiciones con comparación estricta (`===`, `!==`, `>=`, `<=`).
- Combinar condiciones con operadores lógicos (`&&`, `||`, `!`).
- Usar **numeric separators** para cantidades grandes.
- Mostrar resultados claros con `console.log()` y comentarios en español.

---

## 📋 Ejemplo de Cálculos
- **Sepulturas disponibles** = capacidad total – ocupadas.  
- **Costo con descuento** = costo base – (costo base × descuento).  
- **Promedio por bloque** = capacidad total ÷ número de bloques.  
- **Validaciones**: cementerio lleno, descuento especial, acceso permitido.
