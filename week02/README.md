# 📦 Proyecto Semana 2 - Gestión de Cementerios

Este proyecto corresponde al único entregable obligatorio de la **Semana 2** del bootcamp.  
El objetivo es construir un **script en JavaScript** que muestre en consola una ficha de datos de una entidad del dominio **Gestión de Cementerios**, aplicando lo aprendido sobre **variables y tipos de datos**.

---

## 🎯 Objetivos del Proyecto
- Declarar variables con `const` y `let`.
- Usar correctamente los cinco tipos primitivos: `string`, `number`, `boolean`, `null`, `undefined`.
- Identificar tipos con `typeof`.
- Realizar conversiones explícitas con `Number()`, `String()` y `Boolean()`.
- Aplicar reglas de nomenclatura (`camelCase` y nombres descriptivos).
- Usar **numeric separators** (`500_000`) para mayor legibilidad.
- Mostrar la información organizada en consola con separadores visuales.

---

## 📋 Entidad Principal
- **Nombre:** Sepultura Familiar Pérez  
- **Código:** #01  
- **Categoría:** Sepultura doble - Adultos  
- **Capacidad:** 2 cuerpos  
- **Año de construcción:** 1985  
- **Costo base:** 500_000 (con separador numérico)  
- **Estados booleanos:** Disponible (true), Mantenimiento activo (false)
