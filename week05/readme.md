# 📦 Proyecto Semana 5 - Condicionales (Gestión de Cementerios)

Etapa 0 — Fundamentos de Programación | Semana 5 de 10  

Este proyecto corresponde al único entregable obligatorio de la **Semana 5**.  
El objetivo es construir un **script en JavaScript** que controle el flujo de ejecución con condicionales, adaptado al dominio **Gestión de Cementerios**.

---

## 🎯 Objetivos del Proyecto
- Controlar el flujo de ejecución con `if`, `else if` y `else`.
- Simplificar condiciones con el operador ternario `? :`.
- Seleccionar entre múltiples opciones con `switch`.
- Usar `??` (nullish coalescing) y `?.` (optional chaining).
- Distinguir valores *truthy* y *falsy* para condiciones precisas.
- Aplicar operadores lógicos `&&` y `||` como expresiones condicionales.

---

## 📋 Ejemplo de Adaptación al Dominio
Entidad: **Sepultura Familiar Pérez**  
- Validar si la sepultura está disponible (`if/else`).  
- Determinar mensaje de mantenimiento con operador ternario.  
- Clasificar tipo de sepultura con `switch`.  
- Usar `??` para mostrar un valor por defecto si falta información.  
- Usar `?.` para acceder a propiedades opcionales (ej. datos de contacto).  
- Evaluar condiciones con valores truthy/falsy (ej. pago realizado).
