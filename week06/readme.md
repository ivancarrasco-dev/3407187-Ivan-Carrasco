# 📦 Proyecto Semana 6 - Reporte con Bucles (Gestión de Cementerios)

Este proyecto corresponde al único entregable obligatorio de la **Semana 6**.  
El objetivo es implementar un **sistema de reporte** que procese una colección de datos del dominio **Gestión de Cementerios**, usando todos los tipos de bucle vistos en clase.

---

## 🎯 Objetivos del Proyecto
- Definir un array con al menos 6 elementos del dominio.
- Recorrer elementos con `for...of`.
- Contar categorías con `for...of` y contadores.
- Calcular totales y promedio con acumuladores.
- Encontrar máximo y mínimo.
- Generar reporte numerado con `for` clásico.
- Usar `while` o `do...while` en alguna parte del código.
- Incluir `break` o `continue` en un bucle.

---

## 📋 Ejemplo de Adaptación al Dominio
- **Entidad:** Sepultura  
- **Categorías:** Simple, Doble, Infantil, Panteón  
- **Valor numérico:** Costo en pesos
