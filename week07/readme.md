# 📦 Proyecto Semana 7 - Librería de Funciones Reutilizables (Gestión de Cementerios)

Este proyecto corresponde al único entregable obligatorio de la **Semana 7**.  
El objetivo es construir una **librería de funciones reutilizables** aplicada al dominio **Gestión de Cementerios**, usando parámetros, `return`, arrow functions y callbacks.

---

## 🎯 Objetivos del Proyecto
- Definir constantes globales y un array de datos del dominio.
- Crear una función de formato (arrow function).
- Crear una función de cálculo (función pura).
- Crear una función de validación.
- Crear una función con parámetro por defecto.
- Generar un reporte usando bucles y funciones como callbacks.

---

## 📋 Ejemplo de Adaptación al Dominio
- **Entidad:** Sepultura  
- **Función de formato:** `formatSepultura(sepultura)`  
- **Función de cálculo:** `calculateDiscount(price, discountRate)`  
- **Función de validación:** `isAvailable(sepultura)`  
- **Función con parámetro por defecto:** `calculateMaintenanceCost(baseCost, factor = 0.1)`  
- **Reporte:** recorre las sepulturas y aplica las funciones anteriores.
