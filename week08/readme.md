# 📦 Proyecto Semana 8 - Gestión de Inventario con Arrays (Gestión de Cementerios)

Este proyecto corresponde al único entregable obligatorio de la **Semana 8**.  
El objetivo es construir un **script de gestión de inventario** aplicado al dominio **Gestión de Cementerios**, usando métodos de arrays.

---

## 🎯 Objetivos del Proyecto
- Definir un array con mínimo 5 objetos del dominio.
- Usar métodos de mutación (`push`, `pop`, `unshift`, `splice`).
- Usar métodos de búsqueda (`find`, `filter`).
- Usar métodos de iteración (`forEach`, `map`).
- Usar el **spread operator** para copiar o fusionar arrays.
- Generar un reporte final mostrando cambios y resultados.

---

## 📋 Ejemplo de Adaptación al Dominio
- **Entidad:** Sepultura  
- **Propiedades:** id, name, categoría, costo, disponible  
- **Propiedades adicionales:** booleano (`disponible`), numérico (`costo`)
